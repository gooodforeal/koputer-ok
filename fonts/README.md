# Шрифты для PDF генератора

Эта папка предназначена для размещения шрифтов Arial, используемых при генерации PDF.

## Установка шрифтов

1. Скачайте шрифты Arial (если их нет в системе):
   - `arial.ttf` - обычный Arial
   - `arialbd.ttf` - жирный Arial (Bold)

2. Поместите файлы шрифтов в эту папку (`fonts/`)

3. Шрифты должны быть названы:
   - `arial.ttf` или `Arial.ttf` - для обычного
   - `arialbd.ttf` или `Arial-Bold.ttf` - для жирного

## Примечание

- На Windows обычно Arial уже установлен в системе, поэтому добавление шрифтов сюда необязательно
- Если системный Arial не найден, генератор PDF будет использовать шрифты из этой папки
- Убедитесь, что у вас есть права на использование шрифтов Arial
